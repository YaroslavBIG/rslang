## Запуск сборки 

|                                                 |
|:-----------------------------------------------:|
|Установка node modules                           |
|`npm install`                                    |
|Запуск проекта в development mode                |
|`npm run dev`                                    |
|Запуск проекта в  production mode                |
|`npm run prod`                                   |
|Проверка на ошибки eslint                        |
|`npm run lint`                                   |
|Автоисправление ошибок eslint                    |
|`npm run fix`                                    |
